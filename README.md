# Employee Management System

Spring Boot application for managing employee data with CRUD functionality, built using MVC architecture and database integration.

## Technologies
Java • Spring Boot • MVC • Thymeleaf • Maven

## Run
mvn spring-boot:run  
http://localhost:8080
